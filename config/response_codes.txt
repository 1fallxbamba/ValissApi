1. LOGIN

- WPWD : Wrong Password

- UDNE : User Does Not Exist


2. REGISTER

- NUSR : New User Successfully Registered

- URE : User Registration Error (the user data has been inserted in the user_data table but not in the Users table)

- GRE : General Registration Error (the user has not been registered in no table)


3. VALIDATE 

- UNA : Username Not Available

- UIA : Username Is Available
